* Description:
Рецепты по изготовлению программ из текстовых файлов.

Критически относитесь к тексту: в нём гротеск искусно сочетается с
непреднамеренными ошибками.
#+BEGIN_COMMENT
преувеличения с издёвкой
#+END_COMMENT
